1PiecesOfLife — warm, letter-like redesign

A self-hosted private newsletter for a handful of people. The redesign trades the near-black UI for a warm, paper-like surface that feels like opening a letter — calm, intimate, unmistakably a usable app (no marketing landing). Member screens lead; admin screens stay efficient. Pan and zoom freely; each card has an id you can reference in chat (e.g. "make 1c the default editor").

1aDesign system & principles
The system
Warm paper, soft ink, one clay accent, sage for "live" states.

Key UX moves
  • Member-first home. Landing on an open issue shows the writing space, not a dashboard.
  • Autosave you can trust. Draft → Saving… → Saved, with a timestamp.
  • Progress that counts the right people. Admin progress excludes admins by default, with an "I'm participating too" toggle (see 1g).
  • Honest empty states. Every empty view tells you what happens next, not just "nothing here".
  • Mobile is first-class. Single-column, 44px+ targets, sticky save bar (see 1k).
